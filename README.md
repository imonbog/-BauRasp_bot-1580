Исходный код телеграм бота @BauRasp_bot

Содержит следующие функции:
•	Сменить класс – выбор класса для использования других функций
•	Уроки сегодня – получение уроков на сегодняшний день для выбранного класса
•	Уроки завтра – аналогичная прошлой функция; получение уроков на завтрашний день для выбранного класса
•	Информация – получение основной информации о боте, инструкцию по работе с ботом
•	Грядущие события – получение информации о событиях школы, которые должны быть ранее заполнены администратором
•	Расписание по дням – получение расписания на конкретный день для выбранного класса
•	Меню – получение меню столовой школы

Создан учеником школы №1580 Богданом К. (и для школы №1580 :D)

Использует aiogram
Не использует FSM

Создан в рамках школьного проекта «Telegram - бот «Школьный центр» на Python»
